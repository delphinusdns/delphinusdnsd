Delphinusdnsd

Delphinusdnsd is a DNS Server. The first release was made on January 1st, 2016, basing mainly on a program made by the author in November 2005.

The DelphinusDNS Open Source project is going to conclude this year (2024), at which time an Open Core project is going to evolve out of it. Versions beyond 1.9 are not planned before 2026. People interested in continuing development are encouraged to start a fork, or wait up until 2026.
